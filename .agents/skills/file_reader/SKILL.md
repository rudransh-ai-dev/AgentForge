# Skill: File Reader
Description: Allows the agent to read and summarize files in the current workspace.

## Instructions
When the user asks about a file, use the available workspace tools to:
1. List files in the project to locate the target.
2. Read the content of the specific file.
3. Provide a summary or specific details as requested.

## Expected Behavior
- Locate files accurately using partial paths.
- Summarize long files to keep context concise.
- Highlight key functions, classes, and logic.
